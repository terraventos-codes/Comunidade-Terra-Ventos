# 🌬️ Terra Ventos — Landing Page

Landing page oficial da **Comunidade Terra Ventos**, uma rede exclusiva que conecta **investidores, atletas e pessoas do lifestyle do litoral** a **oportunidades imobiliárias selecionadas**, com curadoria jurídica e visão de longo prazo.

---

## ✨ Sobre a Terra Ventos

A **Terra Ventos** nasce no litoral do Ceará como um ponto de encontro entre **capital, pessoas e território**.

A comunidade oferece:

- Acesso antecipado a oportunidades imobiliárias
- Curadoria jurídica e estratégica
- Networking qualificado
- Eventos e experiências exclusivas
- Atendimento consultivo personalizado

---

## 🧩 Estrutura da Landing Page

- **Hero** — proposta de valor e CTAs principais
- **Benefícios Exclusivos** — acesso, segurança e conexões
- **Conexões** — networking e comunidade
- **Fundador** — visão e experiência do projeto
- **FAQ** — dúvidas frequentes
- **Cadastro** — formulário para entrada na comunidade

- **Framer Motion** — animações

## 📦 Instalação

### Pré-requisitos

- Node.js 18+
- npm ou yarn

### Passos

```bash
git clone <url-do-repositorio>
cd landing-terraventos
npm install
npm run dev
```

### Acesse:

👉 http://localhost:3000

### 🔐 Variáveis de Ambiente

- Crie um arquivo .env.local:

```bash
NEXT_PUBLIC_EMAILJS_SERVICE_ID=seu_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=seu_template_id
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=sua_public_key
```
